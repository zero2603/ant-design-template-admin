# ant-design-template-admin
Admin Template with Ant Design

# Các thư viện đặc biệt sử dụng trong project
- Ant Design v4: https://ant.design/
- Ant Chart: https://charts.ant.design/ và https://g2plot.antv.vision/zh/examples/pie/basic ([Link demo ReactJS tại đây](https://charts.ant.design/demos/global))

# Chạy Project

Chạy lệnh `npm start` trong console

# Deployment

- Sửa thuộc tính `homepage` trong package.json thành địa chỉ IP hoặc tên miền bạn muốn deploy. 
Ví dụ: `"homepage": "http://example.com"` hoặc `"homepage": "http://example.com/admin"` (đối với deploy lên subfolder)
- Chạy lệnh `npm run build` trong console
